# The Oracle - Task Management Project

The Oracle is a task management application built with the MERN stack (MongoDB, Express, React, and Node.js).

## Status
Archived.

## Features
- **Task Management**: Create and manage tasks with priority, category, and energy levels.
- **Oracle System**: Suggests tasks based on the user's current energy state.
- **Dashboard**: Visualizes task statistics and energy usage heatmap.
- **UI System**: Custom component-based design with a consistent visual style using Tailwind CSS.
- **Navigation**: Persistent shared layout with a fixed bottom navigation bar.

## Technology Stack
- **Frontend**: React, Tailwind CSS, Lucide React, Framer Motion.
- **Backend**: Node.js, Express, JWT Authentication.
- **Database**: MongoDB.
- **DevOps**: Docker, Docker Compose.

## How to Run

1. Clone the repository.
2. Configure environmental variables in a `.env` file.
3. Start the application using Docker:

```bash
docker-compose up --build
```
